# Summary

Uses gnu-stow to automatically create all the symlinks in the correct
places using the structure of the dotfiles directory.

# Requirements

The only requirement is ``` gnu-stow ```.

# Usage

Clone this repository into anywhre in the filesystem:

```bash
git clone git@github.com:hoomishalom/dotfiles.git
```

Then cd into the ```dotfiles``` directory and run:
```bash
stow .
```
